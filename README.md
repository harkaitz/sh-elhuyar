# SH-ELHUYAR

Euskara<->Gaztelania komando lerro hiztegia UNIX sistemetarako.

```
> elhuyar -h
Usage: elhuyar -v -l es|eu TERM

[eu] Komando lerro interfazea Elhuyar [euskara<->gaztelania]
     hiztegirako.

[es] Interfaz de linea de comandos para el diccionario Elhuyar
     [vasco<->castellano].

[en] Command line interface to the Elhuyar [Basque<->Castillian]
     dictionary.

Bug reports, feature requests to gemini|https://harkadev.com/oss
Copyright (c) 2022 Harkaitz Agirre, harkaitz.aguirre@gmail.com

```
